# Cadastro de Produtos 📦💻
## Introdução 📚

O projeto de Cadastro de Produtos foi desenvolvido em Python com o objetivo de criar um sistema simples para registrar e organizar informações de produtos. A aplicação permite cadastrar dados como nome do produto, preço e quantidade em estoque, proporcionando experiência prática no desenvolvimento de sistemas de gerenciamento.

Durante o desenvolvimento, foram utilizados conceitos importantes da programação, como variáveis, listas, dicionários, estruturas condicionais, laços de repetição e funções, além da interação com o usuário através do terminal.

## Descrição do Projeto ⚙️

O sistema foi criado para facilitar o controle e a organização de produtos de maneira prática e eficiente. O programa permite adicionar novos produtos, visualizar informações cadastradas e manipular dados de forma simples.

Esse projeto ajudou no aprendizado sobre armazenamento de informações, validação de dados e criação de menus interativos, fortalecendo conhecimentos em lógica de programação e organização de código em Python.

## Conclusão ✅

O desenvolvimento do projeto de Cadastro de Produtos contribuiu para aprimorar habilidades em Python e compreender melhor o funcionamento de sistemas de gerenciamento de estoque e informações. Através dele, foi possível praticar conceitos fundamentais da programação, desenvolver soluções organizadas e melhorar a lógica computacional e a resolução de problemas. 🚀🐍
